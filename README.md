# QRCode

## 纯Web前端 HTML / Vue.js ：二维码（动态生成、扫描、识别、解析、扫一扫）

### Demo地址：[https://demo.muguilin.com/二维码生成与识别](https://demo.muguilin.com/二维码生成与识别)

---

> * 该Demo功能共分为：HTML版 和 Vue版(Vue2、Vue3) 两个版本！
>
> * 不依赖任何关于微信环境、微信JS-SDK，微信受权等！
>
> * 支持二维码生成，实时动态生成和渲染！
>
> * 支持扫一扫识别，从相册选择二维码识别！
>
> * 支持PC端，手机移动端等！
